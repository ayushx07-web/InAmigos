# 🌿 InAmigos Foundation Awareness Webpage

A responsive NGO awareness webpage created to spread awareness about the initiatives, projects, and social impact of **InAmigos Foundation**.

This project was developed as part of an internship task to showcase the NGO’s mission and encourage people to volunteer, donate, and support social causes.

---

# 📌 About the Project

The website highlights:

* Introduction about InAmigos Foundation
* NGO mission and purpose
* Ongoing projects and campaigns
* Social impact initiatives
* Volunteer opportunities
* Call-to-action sections

The webpage is designed with a clean and modern UI using **HTML, CSS, and JavaScript**.

---

# 🚀 Features

✅ Responsive Design
✅ Modern NGO Landing Page
✅ Smooth Scroll Reveal Animations
✅ Projects Showcase Section
✅ Volunteer / Join Us Section
✅ Social Impact Statistics
✅ Mobile-Friendly Layout
